🛠️ Hub Industrial - Task Management System
Aplicação desenvolvida para otimização de fluxos de manutenção e ordens de serviço em ambientes industriais. O sistema foca em alta disponibilidade de dados e priorização inteligente de tarefas críticas.

🚀 Key Features
Agrupamento Dinâmico: Organização automatizada por Responsáveis e Áreas.

Lógica de Priorização: Algoritmo que fixa tarefas de Prioridade 0 (Críticas) no topo de cada agrupamento.

Interface Híbrida: Alternância entre Data Table técnica e Kanban-style Cards.

Real-time Sync: Integração com PocketBase para persistência imediata e atualização de status.

Gestão de Configurações: CRUD completo para parâmetros do sistema (Projetos, Áreas e Staff).

🛠️ Tech Stack
Core: Next.js 15 (App Router), React, TypeScript.

Styling: Tailwind CSS (Responsive Design).

Database/Auth: PocketBase (SQLite).

Icons: Lucide React.
